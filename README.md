# 💰 Personal Expense Tracker

A clean, responsive personal finance dashboard built with vanilla HTML, CSS, and JavaScript.

🔗 **Live Demo:** https://kaichengyeoh.github.io/expense-tracker/

---

## Features

- **Add Income & Expenses** — Log transactions with category, description, and date
- **Persistent Storage** — Data is saved in `localStorage` so nothing is lost on refresh
- **Delete Transactions** — Remove any transaction with automatic balance recalculation
- **Spending Overview** — Dashboard cards showing monthly income, expenses, and spending limit progress
- **Transaction History** — View your 10 most recent transactions at a glance
- **Responsive UI** — Works on desktop and mobile screens

---

## Tech Stack

- HTML5
- CSS3 (Grid, Flexbox, animations)
- JavaScript (ES6+, localStorage API)
- Font Awesome (icons)

---

## Getting Started

No installation needed. Just open `index.html` in your browser, or visit the live demo above.

To run locally:
```bash
git clone https://github.com/kaichengyeoh/expense-tracker.git
cd expense-tracker
open index.html
```
